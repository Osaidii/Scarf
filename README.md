# Scarf
My First Published Game  
Free to PLAY!  
You can download this game here or from my itch.io page [osaidii.itch.io](https://osaidii.itch.io)   
This is a platformer about a person with a Scarf!    

Controls  
A / D – Move  
Shift – Run (Very Important)  
Space – Jump  
Q – Dash (Later in game)  
E – Attack (Later in game)  
Esc – Pause  


This source files of the game are use free to use and modify!  
These files are meant to be used in Godot Engine!
